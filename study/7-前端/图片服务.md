# 图片服务
## 图片服务介绍

### 1. 关于 `i.pravatar.cc` 这类服务

**核心概念：**
这类网站不存储真实的用户照片，而是根据您提供的参数（如用户ID、邮箱、尺寸等），**动态生成**一个唯一的、风格一致的虚拟头像。

**主要特点和用途：**

*   **占位符**：在网站或应用开发阶段，当还没有真实用户头像时，用它来填充界面，使设计稿或原型看起来更真实。
*   **唯一性**：同一个参数（如邮箱）总是返回同一个头像，不同参数则返回不同头像。这对于测试用户列表等功能非常方便。
*   **简单易用**：通常只需要在URL后面加上一个标识符或尺寸参数即可。
    *   例如：`https://i.pravatar.cc/300?img=5` 会生成一个固定编号为5的300x300像素头像。
    *   例如：`https://i.pravatar.cc/150?u=zhangsan@email.com` 会根据这个邮箱地址生成一个唯一头像。
*   **保护隐私**：由于是虚拟头像，避免了在测试环境中使用真实用户照片可能带来的隐私问题。
*   **风格统一**：生成的头像通常具有统一的艺术风格（如卡通、抽象、写实等），让界面看起来更协调。

**同类服务举例：**

*   **UI Avatars**: 生成带字母的简约头像。
*   **DiceBear**: 提供多种艺术风格（像素、标识、卡通等）的卡通头像。
*   **Adorable Avatars**: 较早的虚拟头像服务，生成卡通怪物头像。
*   **Lorem Picsum**: 使用随机的真实风景照片作为占位图，而不是头像。

### 2. 如何找到某一个领域的特定图片

如果您想找的不是虚拟头像，而是真实的、特定领域的图片（例如“自然风光”、“办公室工作”、“医疗科技”等），您需要使用专业的**图片库/图库网站**。

这些网站主要分为两大类：**免费图库**和**付费图库**。

#### A. 免费图库网站

这些网站提供的图片通常遵循 **CC0（知识共享零）** 许可，意味着您可以免费用于个人甚至商业项目，无需署名（但阅读具体许可条款是很好的习惯）。

**推荐网站：**

1.  **Unsplash**： 质量极高，艺术感强，摄影师社区非常活跃。是设计师和开发者的首选。
2.  **Pexels**： 图片和视频质量都很高，搜索功能强大，分类清晰。
3.  **Pixabay**： 内容非常丰富，包括图片、矢量图、插图和视频。
4.  **Freepik**： 主要以矢量图、PSD模板和图标为主，也有大量照片。部分内容需要署名。
5.  **Burst（by Shopify）**： 专注于为电商和商业网站提供免费图片。

**使用技巧：**
*   使用**英文关键词**搜索通常能得到最广泛和最准确的结果。
*   善用网站提供的**分类、合集和色彩筛选**功能。

#### B. 付费图库网站

这些网站提供专业级、高质量的图片、插图和视频，内容更全面，但需要付费购买或订阅。

**推荐网站：**

1.  **Shutterstock**： 全球最大的图库之一，内容海量，几乎能找到任何主题的图片。
2.  **Getty Images**： 以高端、编辑类（新闻、娱乐）图片闻名，质量极高，价格也相对昂贵。
3.  **iStock**： 隶属于Getty，提供性价比更高的图片购买方案。
4.  **Adobe Stock**： 与Adobe Creative Cloud（如Photoshop, Illustrator）无缝集成，对设计师来说非常方便。

**使用技巧：**
*   可以先在免费图库中寻找，如果找不到满意的，再考虑付费图库。
*   付费图库通常提供不同大小的图片和授权选项，购买前请确认您需要的用途。

### 总结

| 需求类型 | 推荐工具 | 举例 |
| :--- | :--- | :--- |
| **开发测试、虚拟用户头像** | **占位头像生成器** | `i.pravatar.cc`, UI Avatars, DiceBear |
| **寻找真实、高质量的特定领域图片** | **免费图库** | Unsplash, Pexels, Pixabay |
| **寻找专业、独家、海量的特定领域图片** | **付费图库** | Shutterstock, Getty Images, Adobe Stock |
